# A Rush Hour Web-app

Includes a solver.

## Usage:
### Via Command Line Interface:

```sh
php rushHourCli.php < input-board.txt
```

Example for `input-board.txt`:
```
....AA
..BBCC
rr..EF
GGHHEF
...IEF
...IJJ
```

### Via Web:
1. Set up your web server to execute `public_html/api.php`
2. Access the api via the web
